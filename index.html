Check-Up Me (MedGO)

Interactive Diagnostic Web Platform

A robust, web-based healthcare prototype engineered to bring medical assistance to users' fingertips. Built as part of a university initiative, the platform aims to improve overall health through the early detection of diseases.

The platform features a symptom-based diagnostic tool, a comprehensive A-Z Medical Encyclopedia, and a geolocation-based "Find a Doctor" interface. The UI is fully responsive, leveraging modern front-end technologies to deliver a fast, reliable, and accessible user experience across all devices.

HTML5 CSS3 / Animations JavaScript Responsive Web Design
